Setup Hexo for GitHub Page in either Windows or WSL

2018-10-13
SoftwareWebsite
GitHub, Hexo, Windows

This post talks about how to setup Hexo in Windows and host a site in GitHub.

Install tools for Hexo

  • Download and install lastest version of Node.js.

  • Install git if you havenâ€™t done so.

  • Install Hexo:
    npm install hexo-cli -g
    hexo init blog
    cd blog
    npm install
    hexo g # or hexo generate
    hexo s # or hexo server
    hexo d # or hexo deploy

  • Create a new post
    hexo new "My New Post"
